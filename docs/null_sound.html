    Luv.Audio.NullSound

    This class has the same methods as Luv.Audio.Sound, but they do nothing; they don't play or try to load any sounds.

    Browsers which don't support the <audio> tag will instantiate this class instead of Luv.Audio.Sound.

    Luv.Audio.NullSound = Luv.Class('Luv.Audio.NullSound');
